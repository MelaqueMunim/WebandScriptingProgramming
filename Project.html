My Work

GUI Scheduler application Project

In this project, the program provides a user-friendly interface for visualizing CPU scheduling algorithms. Users begin by launching the program, selecting an input file, and then choosing from a variety of scheduling algorithms, including Non-Preemptive Highest Priority First (HPF), First Come First Served (FCFS), Round Robin with a fixed time quantum (RR), and Preemptive Shortest Remaining Time Next (SRTN). By clicking the "Show/Update Graph" button, users can generate visual representations of these algorithms' schedules.

Python Hang-Man Project

In this project,I and a team Python hangman game lets players guess a secret word one letter at a time. It starts with a menu for playing or exiting. During the game, players input letters, which are checked for validity and repetition. The game updates the word display and tracks incorrect guesses. It ends when players win, lose due to too many mistakes, or exit. Afterward, the player gets the result, and the secret word is revealed.
